A powerful WordPress slider plugin for Gutenberg block editor powered by Swiper.js. Create stunning, fully responsive sliders with advanced animation effects, customizable navigation, and professional features.

== Description

Smart Block Slider Pro is a premium extension that enhances the Gutenberg Slider plugin with advanced features and professional capabilities. Built on top of the free Gutenberg Slider plugin, this Pro version unlocks powerful animation effects, extended customization options, and enterprise-level functionality.

Note: This plugin requires the free Gutenberg Slider plugin to be installed and activated. The Pro version extends the free version with additional features and customization options.

== Key Features ==

⭐ 12+ Animation Effects – Choose from slide, fade, flip, cube, cards, coverflow, shadow push, zoom split, slide flow, flip deck, twist flow, and mirror effects
⭐ Advanced Navigation – Multiple arrow styles (5 built-in styles + custom option) with full customization
⭐ Thumbnail Gallery – Display thumbnail gallery below the main slider with customizable size and spacing
⭐ Scrollbar Navigation – Customizable scrollbar with position options (top, bottom, left, right) and color control
⭐ Vertical Slider – Create vertical sliders with full height control
⭐ RTL Support – Right-to-left language support for international websites
⭐ Grid Layout – Enable grid layout with row or column options for multi-slide displays
⭐ Image Zoom – Pinch-to-zoom functionality for images on touch devices
⭐ Keyboard Control – Navigate sliders using arrow keys
⭐ Mousewheel Control – Scroll through slides using mouse wheel
⭐ Slides Grouping – Control how many slides advance at once
⭐ Coverflow Options – Advanced coverflow settings (rotate, stretch, depth, modifier, shadows)
⭐ Centered Slides – Center slides for better visual presentation
⭐ Slides Per View Auto – Automatic slides per view calculation
⭐ Fully Responsive – Mobile-first design that works on all devices
⭐ Gutenberg Block Editor – Native integration with WordPress block editor
⭐ Developer-Friendly – Extensible with filters and actions for customization

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/block-slider/` directory or install it via **Plugins > Add New**.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. Navigate to **Smart Block Slider** in the WordPress admin menu to create a new slider.
4. Add slides using the Gutenberg block editor - each slide can contain any combination of blocks (images, text, videos, etc.).
5. Configure slider options in the settings tabs: Transition, Navigation, Pagination, Autoplay, Responsive, Scrollbar, Thumbnail Gallery and Other Options.
6. Use the shortcode `[wpsbs_slider id="SLIDER_ID"]` to display the slider anywhere on your site, where `SLIDER_ID` is the ID of your slider.

== Frequently Asked Questions ==

= Do I need the free version to use this plugin? =
Yes. Smart Block Slider Pro is an extension that requires the free Smart Block Slider plugin to be installed and activated. The Pro version adds advanced features on top of the free version.

= What transition effects are available? =
The Pro version includes 10+ transition effects.

= Can I customize the navigation arrows? =
Yes! The Pro version offers 5 built-in arrow styles plus a custom option. You can fully customize arrow appearance, position, and behavior.

= Can I add a thumbnail gallery? =
Absolutely! Enable the thumbnail gallery feature to display clickable thumbnails below your main slider. Customize thumbnail size, spacing, and loop behavior.

= Can I use keyboard navigation? =
Yes. Enable keyboard control to navigate slides using arrow keys on your keyboard.

= Is the plugin responsive? =
Absolutely! All sliders are fully responsive and mobile-friendly, automatically adapting to different screen sizes.
